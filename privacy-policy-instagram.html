1. 管理者情報

2. 収集する情報

  • Instagram / Facebook から取得する情報：ビジネスアカウントの基本情報（ユーザー名、プロフィール、メール等）、投稿メタデータ（APIで許可された範囲）
  • アプリで生成・保持するデータ：アクセストークン（安全に管理）、ログ、設定

3. 利用目的

  • Instagram アカウントを用いたログイン認証
  • 投稿情報・インサイトの取得（ユーザーが許可した範囲）
  • サービスの提供・改善
  • 不正利用の防止およびセキュリティ管理

4. 第三者提供

法令に基づく場合を除き、ユーザーの個人情報を無断で第三者提供することはありません。Instagram API を通じて取得した情報は、本アプリの機能提供のために利用します。

5. 保有期間

情報はサービス提供に必要な期間のみ保持します。保持期間の目安や削除希望はお問い合わせください。

6. ユーザーの権利

ユーザーは自身の情報の確認、訂正、削除を請求できます。請求方法については運営者までご連絡ください。

7. セキュリティ

アクセストークン等の機微情報は暗号化して保存し、アクセス権限を限定します。万一の漏洩が疑われる場合は速やかに通知します。

8. 外部サービス

本アプリは Instagram / Facebook の API を利用します。これらのサービスの利用にあたっては、各プラットフォームのポリシーにも従います。

9. 改定

本ポリシーは必要に応じて改定します。重要な変更は当ページで告知します。
